CF704B 题解

文章发布时间:
2023-05-02

最后更新时间:
2023-02-14

这是道充分体现了水淹笛卡尔树这个模型的应用的好题。

类似题目：洛谷 P5999，ARC117E

观察这题的权值函数，与排列的相邻元素有关，与元素的大小关系有关。

而笛卡尔树，它的中序遍历记录了位置相关信息，它的拓扑序记录了权值相关信息。

而水淹笛卡尔树，就是一类在笛卡尔树上按层/权值 DP 的思路。

往往我们并不需要显式地说明笛卡尔树的构型，也有些时候，这类题目可以不用笛卡尔树来理解。

既然是按权值 DP ，我们考虑从小到大插入每个数 \(i\)。

在 \(i\) 之前，已经插入的数会形成若干个连续段，这些数的贡献是已经计算了的，我们只需知道 \(i\) 与两侧数的大小关系，即可计算 \(i\) 的贡献。

在写方程之前，重写一下题目所给的量：\(a_i \gets a_i + x_i,b_i \gets b_i - x_i,c_i \gets c_i + x_i,d_i \gets d_i - x_i\)。

那么 \[f(i,j) = \begin{cases} c_i + b_j ,i > j \\ d_i + a_j,i < j \\ \end{cases}\]

考察 \(i\) 的转移：

先考虑 \(i \ne s,t\) 的情况：

  1. \(i\) 把已有的两段接起来，在 \(j > 1\) 时转移，显然 \(i\) 两侧的数小于 \(i\)，那么 \(i\) 的贡献就是 \(a_i + c_i\)，则有转移 \(f_{i,j-1} \gets f_{i-1,j} + a_i + c_i\)。

  2. \(i\) 新开一段，那么它周围的两个数都大于它，在 \(j = 1\) 且 \(i > s,t\) 时无法转移：

    \(f_{i,j+1} \gets f_{i-1,j} + b_i + d_i\)。

  3. \(i\) 接在某一段的左边，在 \(j = 1\) 且 \(i > s\) 时无法转移：

    \(f_{i,j} \gets f_{i-1,j} + c_i + b_i\)。

  4. \(i\) 接在某一段的右边，在 \(j = 1\) 且 \(i > e\) 时无法转移：

    \(f_{i,j} \gets f_{i-1,j} + a_i + d_i\)。

在 \(i = s\) 或 \(i = t\) 时，新开一段或接在某一段边上即可。
