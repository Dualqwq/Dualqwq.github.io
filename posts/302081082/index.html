ARC146E 题解

First Post:
2023-05-02

Last Update:
2023-02-14

因为之前做过水淹笛卡尔树的题（ARC117E），一眼看出了 DP 状态，后面就不会了，题解告诉我第二维有效状态不超过 $5$ 个，人有点麻。也是没有仔细思考的缘故。

按值域从小到大 DP，每次考虑插入 $A_i$ 个 $i$。

假设已经插入的数形成了 $j$ 个连续段，显然，除了最左和最右的连续段，剩下的段的两端都是 $i - 1$（不然就无法满足相邻位置差为 $1$）

这提示我们最左边和最右边有一些特殊性。

事实上，如果最左边是 $i - 1$，那么当前的 $i$ 还可以插到左边，否则不行。右边同理。

那么我们设 $dp_{i,j,u,v}$ 表示当前做到 $i$，一共形成了 $j$ 个连续段，最左边是不是 $i$，最右边是不是 $i$。

转移有两个方面，第一是 $j$ 的变化，第二是转移系数。

如果我们想在最左和最右都填上 $i$ 的话，我们首先会把 $2j$ 个 $i$ 填在每个连续段的两侧，然后剩余的 $A_i -2j$ 个 $i$ 会贡献 $A_i - 2j$ 个连续段，总段数就是 $A_i - j$。

如果想在某几侧不填上 $i$ 的话，道理是一样的，将结果 $+1$ 或 $+2$ 即可。

考察转移系数，还是以 $u = v = 1$ 为例，这时一共会有 $j + 1$ 个空隙，每个空隙都必须有数，这就是个插板法，方案数为 $\binom{A_i - 1}{j}$。

直接这么做，看上去可以达到平方复杂度，但是，我们发现 $j$ 的变化方法较为单一，它只有可能在每轮中由 $j$ 变为 $A_i - j$，或在 $u$ 或 $v$ 变化时加上 $1$ 或者 $2$。

我们处理出 $cur_i$ 表示如果前 $i$ 轮，每轮都在两边放数，会有几个连续段（显然有 $cur_i = a_i - cur_{i-1}$）。

那么 $j$ 的取值只会在 $cur_i$ 附近，取 $[cur_i - 2,cur_i + 4]$ 即可。
