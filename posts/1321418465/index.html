这题我观察出了一些显然的性质，但还是没有做法。（

做完之后，对 SAM 的运用倒是更加熟练了。那个 DP 状态，虽然见过很多回这样的形式了，但还是没想到。

为方便叙述，我们将原串翻转，限制变为 \(t_i\) 是 \(t_{i+1}\) 的真子串。

性质 1

\(|t_i| = i\)，即每加一个串，顶多在开头或结尾加一个字符，这是显然的，因为加得越多，后面的串越难合法。

性质 2

设 \(f_i\) 表示求解 \(s[1 \dots i]\) 中的问题，且强制让 \(t_k\) 的右端点为 \(i\) 时最大的 \(k\)。（实际上就是 \(t_k\) 最大的长度）

那么 \((i -1) - f_{i - 1} + 1 \le i - f_i + 1\) （这里比较的是两个 \(t_k\) 的左端点的位置，事实上，\(f_{i}\) 所代表的方案，每个串都删一个字符，就能变为 \(f_{i-1}\) 的方案，即 \(f_{i - 1} \ge f_i - 1\)，故上述性质显然成立。）

这里的 \(f_i\) 实际上就是 DP 状态，我们考虑维护 \(t_k\) 所代表的区间 \(s[l,r]\)，由性质 \(2\) 可得 \(l\) 是不降的，这启示我们可以双指针。

如何判断一个串 \(s[l,r]\) 是否合法（即成为某个方案中的 \(t_k\) ）呢？

根据意义，我们要找到一个串 \(s[l',r']\) 满足 \(r' < l\) 且 \(s[l',r']\) 为 \(s[l,r]\) 去掉开头或结尾且 \(s[l',r']\) 合法。

此外，若 \(s[l,r]\) 合法，则 \(s[i,r]\) 合法 \((l \le i \le r)\)。

考虑使用一个 SAM 维护右端点 \(< l\) 的合法串的贡献。设在 SAM 上 \(s[l,r]\) 所在的结点为 \(p\)。

那么在双指针的过程中，执行如下算法：

  1. 设 \(g_p\) 表示节点 \(p\) 所包含的最长合法子串的长度。
  2. 当 \(r \gets r + 1\) 时，检查当前的 \(s[l,r]\) 是否合法，即 \(s[l,r - 1]\) 和 \(s[l + 1,r]\) 在 SAM 上的点的 \(g\) 的较大值是否 \(\ge r - l\)。
  3. 如果 \(s[l,r]\) 合法，那么令 \(f_i = r - l + 1\)，令 \(N_i\) 为该串在 SAM 上的节点 \(p\)。并回到第二步。
  4. 如果 \(s[l,r]\) 不合法，那么我们需要在 SAM 上更新 \(g\) 的信息，这个时候 \(N_i\) 就有用了，因为我们已经将 \(l\) 开头的合法串信息存在了 \(N_l\) 里。我们先让 \(g_{N_l}\) 与 \(f_l\) 取 \(\max\)，再跳 \(N_l\) 的父亲 \(t\)，这个时候 \(g_t\) 应该变为 \(len_t\)。如果当前的 \(g_t\) 不为 \(len_t\)，就将其置为 \(len_t\)，否则可以直接 break，因为如果 \(g_t = len_t\)，\(t\) 的祖先 \(a\) 也必然满足 \(g_a = len_a\) 了。做完上述过程后，令 \(l \gets l + 1\)，回到第三步。

另外提一嘴，维护 \(s[l,r]\) 的节点 \(p\)，只需要在 \(r\) 增大时让 \(p\) 沿着 DAWG 向下走一步，在 \(l\) 增大时，如果 \(len_{fa_p} \ge r - l + 1\)，就令 \(p = fa_p\) 即可。

关键在于这个 DP 状态，非常巧妙地把题目中各类性质和对象统一了起来。
