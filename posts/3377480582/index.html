ARC135E 题解

First Post:
2023-05-02

Last Update:
2023-04-22

不知道如何评价题。

题意：

给出两个数 $n,x$，构造序列 $A$，使其满足下列限制：

  1. $A_1 = x$。

  2. $A_i < A_{i+1}$。

  3. $A_i$ 是 $i$ 的倍数。

在此基础上，使得 $\sum A_i$ 最小，输出答案对 $998244353$ 取模的值。

$T$ 组数据。$1 \le T \le 10,1 \le n,x \le 10^{18}$。

显然，策略是让每个 $A_i$ 都尽量小。

那么设 $B_i = A_i / i$，就有 $B_{i+1} = \lfloor \dfrac{A_i}{i+1} \rfloor + 1 = \lfloor \dfrac{iB_i}{i+1}\rfloor + 1 = B_i + 1 + \lfloor{\dfrac{-B_i}{i+1}}\rfloor = B_i + 1 - \lceil \dfrac{B_i}{i+1} \rceil$

因为 $B_i$ 约等于 $\dfrac{X}{i}$ ，设 $t = \dfrac{B_i}{i+1}$，那么会有 $t(i+1)i = O(X)$。

那么可以说明，在 $i > X^{\frac{1}{3}}$ 的时候，$t = O( X^{\frac{1}{3}})$。

也就是说，$\lceil \dfrac{B_i}{i+1} \rceil$ 只有 $O(X^{\frac{1}{3}})$ 个本质不同的值！

因为这玩意单调不增，所以最终会呈现若干个连续段。

然后我想到这里不会算端点！

考虑一个 $l$，设 $t = \dfrac{B_l}{l+1}$ ，我们想求出最大的 $r$ 满足 $x = \lceil\dfrac{B_l - (r - l)(x-1)}{r+1} \rceil$

化式子：
$$
\begin{aligned}
x = \lceil \dfrac{B_l - (r + 1)(x - 1) + (l + 1)(x-1)}{r+1} \rceil \
2x - 1 = \lceil \dfrac{B_l + (l+1)(x-1)}{r+1}\rceil \
r + 1 = \lfloor \dfrac{B_l + (l+1)(x-1) - 1}{2x-2} \rfloor
\end{aligned}
$$

事实上，上文中的 “$B_i$ 约等于 $\dfrac{X}{i}$ “ 是可以严谨的给出一个 上界的。

因为 $A_i \le X + \sum_{j=2}^i j < X + \dfrac{1}{2}i(i+1)$

所以 $B_i \lt \dfrac{X}{i} + \dfrac{1}{2}(i+1)$。

设 $L = X^{1/3}$。

由此可以算出 $\lceil \dfrac{B_L}{L+1} \rceil - 1$ 处于 $O(L)$ 级别，即在 $i > L$ 的时候，$B_i - B_{i+1}$ 的取值也只有 $O(L)$ 个。

于是总段数就是 $O(L)$ 的了。
