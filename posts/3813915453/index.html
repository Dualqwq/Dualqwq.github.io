ARC117E 题解

文章发布时间:
2023-05-02

最后更新时间:
2023-05-02

这题让我没什么头绪。想到过使用前缀和来转换，但 \(\pm 1\) 的变化量看上去非常难以处理。最后发现是一个从未见过的 DP 模型，果然 ARC 题还是有学习价值的（前提是找得到好题）。

题意：计数满足下列条件的长度为 \(2n\) 的序列 \(a_1,a_2,\dots,a_{2n}\)：

  1. 恰好包含 \(n\) 个 \(+1\) 和 \(n\) 个 \(-1\)。
  2. 恰好有 \(K\) 对 \((l,r) (1 \le l \le r \le n)\) 满足 \(a_l + a_{l+1} + \dots + a_r = 0\)

\(1 \le n \le 30,1 \le K \le n^2\)。

求出 \(a\) 的前缀和 \(S\)，设 \(O_i\) 表示有多少个 \(S\) 等于 \(i\)，那么上述条件等价于：\(S_0 = S_{2n} = 0,\sum_i \binom{O_i}{2} = K\)。

另一方面，\(S_i - S_{i + 1} = \pm 1\)。

注意到这个 \(K\) 的约束是关于每种值独立的，我们尝试设计关于值的 DP。

要把值的限制和相差 \(\pm 1\) 的限制统一起来，我们尝试 DP 这个序列 S 的笛卡尔树。

如下图：

我们在这个笛卡尔树上，从上到下，一层一层的 DP,每次填完一层的所有数。

设 \(dp_{i,j,k,c}\) 表示一共填了 \(i\) 个数，已经填的数在 \(S\) 序列上形成 \(j\) 个连续段，一共填了 \(k\) 层，当前的 \(\sum \binom{O_i}{2} = c\) 的方案数。

考虑最后怎么求答案，因为最后强制 \(S_0 = S_{2n} = 0\)，我们考虑把 \(S \lt 0\) 和 \(S \ge 0\) 的部分拼接起来。

注意到这两种情况是对称的，我们都可以用这个 DP 状态来表示。

设 \(g_{i,j,c} = \sum_k dp_{i,j,k,c}\)。

具体地，因为拼的时候肯定是一段 \(S \ge 0\) 和一段 \(S < 0\) 交错，那么我们枚举 \(S \ge 0\) 的个数 \(i\)，段数 \(j\)，这一部分的 \(\sum_i \binom{O_i}{2} = c\)，那么最后的答案就是 \[ \sum_{i,j,c} g_{i,j,c}g_{2n + 1 - i,j - 1,K - c} \] 进一步地，我们发现 DP 中的 \(k\) 是没用的，可以去掉。

我们考虑新一层加入了 \(x\) 个数的影响。

会出现一下三种情况：

  1. 某两段之间的间隙被连接，需要占用一个位置，并把两个段合并成一个
  2. 某个段间隙没有被连接，因为 \(S\) 的变化连续，必须在这个间隙的两段贴两个数来拓展
  3. 加入一个新的“山谷”

重点在于考虑段数的变化。

当前有 \(j\) 个连续段，就有 \(j + 1\) 个空隙，每个空隙中都必须填数（因为我们是从上到下来填的数，每次填的是当前序列中的最小值，而我们限制了序列两端是 \(0\)，所以两端的空隙也必须填数；另一方面，因为 \(S\) 的变化是连续的，所以中间的空隙不填数就不合法了）

由上面三种情况整合可得，在单一个中间空隙中填入了 \(a\) 个数，会给总段数带来 \(a - 2\) 的变化量。两端则是 \(a - 1\)。

整理式子可得新的连续段数为 \(x - j\)。

考虑转移系数，相当于在 \(j +1\) 个盒子里放 \(x\) 个球，每个盒子都不为空，插板可得系数为 \(\binom{x - 1}{j}\)

综上所述，有转移 \(dp_{i + x,x - j,c + \binom{x}{2}} \gets \binom{x - 1}{j} dp_{i,j,c}\)。
