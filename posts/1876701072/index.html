遇到新题型，茫然无措是必然的，但只有冷静下来，运用自己掌握的思维模式，才能从茫然中找到柳暗花明。

考虑已经确定了本体和分身完成的任务集合，怎么判定合法性。

设本体完成的任务为 A 类，分身完成的任务为 B 类。

把 \((t,x)\) 看成二维平面上的点，容易绘制以下图像：

我们的行动轨迹需要经过上图所有的点。

A 类任务可以限制人的行动轨迹，相比 B 类任务具有更加”稳定“的约束，我们考虑研究两个 A 任务之间的间隔。

右图是两个 A 夹着一段极长的 B ，左图是一段极长的 A。

蓝色是不管 B 的路线，红色是管 B 的路线。

容易发现，一段极长的 A，只会贡献一个 B，一段极长的 B 的非开头部分 只会由夹着这一段的两个 A 解决（开头那个被绿色圈住的部分可能已经被前一段解决）

（上述图片来自 @command_block，感谢 cmd 的题解）

两个 A 之间的间隔，就是一段 B 的极长连续段。

假设这些 \(B\) 任务按时间顺序为 \(p_1,p_2,\dots,p_k\)，\(p_i\) 对应的时间区间为 \([l_i,r_i]\)，其实就是 \([t_{p_{i-1}},t_{p_i}]\)。

设初始时间为 \(t\)，初始位置为 \(x\)，假设下一个任务为 \(p_i\)，那么如果 \(t + | x - x_{p_i} | > r_i\)，则方案不合法，否则 \(t\) 变为 \(\max(l_i,t + |x - x_{p_i}|)\)，\(x\) 也对应变化（对 \(l_i\) 取 \(\max\) 是因为，假如我在 \(t_{p_{i-1}}\) 之前就到达了 \(x_{p_i}\)，我也必须等到 \(t_{p_{i-1}}\) 的时候才能撤销原本放在那里的分身，把它移到当前位置）

观察上述的贪心算法，我们只需在 \(dp\) 的时候维护上面的 \(t\) ，然后一个一个加入 B 类任务即可。

具体地，设 \(dp_{i,0/1}\) 表示任务 \(i\) 是 B 类，任务 \(i\) 是/否已经被 \(i - 1\) 之前的某一段 A 解决。（也就是那个绿色圈的情况）

对于 \(f_{i,0}\) ，当前本体肯定在 \(x_i\)，对于 \(f_{i,1}\)，\(i - 1\) 是 A 类，并且本体在 \(x_{i-1}\)。

记 \(g_{l,r}\) 表示任务 \(l \sim r\) 是否能成为一段 A。

记 \((a,b) = |x_a - x_b|\)。

对于 \(f_{i,0}\):

  • \(i - 1\) 是 B 类

    1. 从 \(f_{i-1,0}\) 转移：\(f_{i,0} \gets \max(f_{i-1,0} + (i - 1,i),t_{i-1}) ,(f_{i-1,0} + (i - 1,i) \le t_i)\)
    2. 从 \(f_{i-1,1}\) 转移：\(f_{i,0} \gets \max(f_{i-1,1} + (i - 2,i),t_{i-1}),(f_{i-1,1} + (i - 2,i) \le t_i)\)

  • \(i - 1\) 是 A 类

    枚举 A 类极长连续段 \([j + 1,i - 1]\)。

    \(f_{i,0} \gets t_{i - 1} + (i - 1,i)\)

    条件：

    \(g_{j + 1,i - 1}\) 为真

    且 \(\min(f_{j,0} + dis(j,j + 1),f_{j-1,1} + dis(j - 1,j + 1)) \le t_{j+1}\) （\(j\) 可以走到 \(j + 1\)）

    且 \(t_{i-1} + (i - 1,i) \le t_i\) （可以从 \(i - 1\) 走到 \(i\)）

对于 \(f_{i,1}\)：

\(f_{i,1} \gets t_{i-1}\)

条件：

枚举 A 类极长连续段 \([j + 1,i - 1]\)。

存在一个 \(j\) 满足：

  • \(g_{j +1,i - 1}\) 为真

  • 且 \(\min(f_{j,0} + dis(j,j + 1),f_{j-1,1} + dis(j - 1,j + 1)) \le t_{j+1}\) （\(j\) 可以走到 \(j + 1\)）

  • 且存在一个 \(k \in [j + 1,i - 2],(i,k) + (i,k + 1) \le t_{k + 1} - t_k\) （存在一个空隙使得人可以拐出来，先放好分身再走回去） 或者 \(\max(\min(f_{j,0} + dis(j,i),f_{j,1} + dis(j - 1,i)),t_j) + dis(i,j + 1) \le t_{j+1}\) （这一段 A 的开头和上一段 B 的结尾之间有个空隙可以让人出来放分身）

总的来说，是一道比较复杂的 DP 题，非常具有 CF 风格，虽然最后的转移情况很多，但并没有思维上的难度，真正的难点主要在前面的状态设计和对问题的刻画。注意到 A 类和 B 类各自的贡献条件是十分重要的，这样才能开始后面的转化。

愈是复杂，愈要冷静。
